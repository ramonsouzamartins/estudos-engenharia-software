# Lógica de Programação — Exercícios Iniciais

Este diretório contém exercícios básicos de lógica de programação,
desenvolvidos com o objetivo de praticar raciocínio lógico e estrutura
de código.

## Conteúdos abordados
- Entrada e saída de dados
- Condições (if/else)
- Laços de repetição
- Operações matemáticas simples

## Linguagem
- Python
