### Teste real
Testes do Código
O código foi testado utilizando o aplicativo Swifty Compiler em meu dispositivo IOS..  
Durante os testes, o código não funcionou conforme esperado.
Ao iniciar o processo de correção, foi identificado que o aplicativo possui um limite

o que impediu a conclusão da validação completa do código.  
Além disso, foi possível perceber que existem vários erros no meu código,
que ainda precisam ser analisados e corrigidos em testes futuros.  
> Observação: a correção completa será realizada assim que for possível retomar os
testes sem a limitação da ferramenta, para isso vou tentar instalar alguma ferramenta 
de programação no meu laptop e continuar a correção.
