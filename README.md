# Estudos em Engenharia de Software
Repositório criado para organizar estudos e projetos iniciais em programação
durante o bacharelado em Engenharia de Software.
## Conteúdos
- Lógica de programação
- Exercícios práticos
- Projetos simples para aprendizado
## Objetivo
Desenvolver base sólida em programação e boas práticas de software,
com foco em estágio na área de desenvolvimento.
